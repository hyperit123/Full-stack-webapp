## Jack Knife - Digital Character Sheet
A web-based character sheet for the Jack Knife sci-fi tabletop RPG. It's mad so that those that are intrested in jack knife or just want to try it out for them selves

## Features
- **Character Management**: Create and edit characters with name, gender, race, class, and background
- **23 Unique Races**: Choose from organic or mechanical species, each with unique traits
- **13 Classes**: Gunslinger, Mechanic, Medic, Technomancer, and more
- **Stat System**: 9 core stats (WS, BS, Tn, Str, Dex, INT, PER, WP, FEL) with point-buy allocation
- **Dynamic Health Tracking**: 
  - Wounds based on Toughness
  - Stamina based on Dexterity
  - Interactive checkboxes for tracking
- **Tabbed Interface**: 
  - Actions & Bonus Actions
  - Traits, Resistances, Immunities, Weaknesses
  - Inventory management
- **Currency System**: Track Credits, Doch, and Renown
- **Level Progression**: Automatic point calculation based on character level

## Usage
1. 
2. Click the esc to get out of the overly
3. Allocate stat points (9 base + 1 per level)
4. Add custom actions, traits, and inventory items
5. Track wounds and stamina with checkboxes during gameplay
6. Add custom check box things for othere abilitys like a portal gun with a set amount of uses
